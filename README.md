# Ember-cli-emflux

Flux library for Ember

## Installation

`npm install --save-dev ember-cli-emflux`

## Running

* `ember server`
* Visit your app at http://localhost:4200.

## Running Tests

* `ember test`
* `ember test --server`

## Building

* `ember build`

For more information on using ember-cli, visit [http://www.ember-cli.com/](http://www.ember-cli.com/).
